MetaPrompt

MetaPrompt is a browser extension I helped build for MHacks 2025 that turns highlighted text from any webpage into an effective prompt for the LLM of your choice.

The project uses a self-hosted FastAPI backend with prompt building handled by Gemini. This feature was eventually integrated by Google into Chrome, although we did it first!
