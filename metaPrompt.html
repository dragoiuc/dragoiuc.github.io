MetaPrompt

MetaPrompt is a browser extension I helped build for MHacks 2025 that turns useful text from any webpage into a stronger prompt for the LLM of your choice. Instead of copying context, switching tabs, and rewriting the same setup instructions over and over, users can highlight text, choose a prompt type, add project-specific tags, and generate a cleaned-up prompt that is ready to copy or send into tools like ChatGPT, Gemini, Claude, and Fetch.ai.

The project connects a lightweight extension interface with a FastAPI backend powered by Gemini. The backend takes the selected page context and user options, restructures them into a more specific and useful prompt, and returns a result designed for quick iteration. I focused on making prompt engineering feel less like a separate chore and more like a natural part of reading, researching, and building on the web.
